Daily Text Script

Description

  • This package is a simple command line tool that fetches the Daily Text at 5:30 AM each morning from JW.org then saves it to /Library/DailyText/DailyText.txt. simply type txt to display the text!

    I also recommend downloading QuickActivator and the JW.org Glyph for it and set up an activator listener to run txt so you can quickly get the daily text from your control center. This is shown in the video below
