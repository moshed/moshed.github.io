Dyneppelin

Description

  • Dyneppelin is an attempt at allowing Zeppelin to use more than one image. However it only works with images that are in the logo@2x.png format, which means they will only appear as black or white and alpha. Also it only updates to a new image when closing or opening an app, and going to or from lock screen. It does however work! So I would like if you gave it a try! You can do as many images as you'd like and set it to a custom time in seconds for it to change. All it is, is a bash and Python script, and a launch daemon.
